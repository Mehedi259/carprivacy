### Step 3: App Store Connect এ Add করুন 🍎
1. **App Store Connect** এ login করুন
2. **Auto Intel** app select করুন
3. **App Privacy** section এ যান
4. **Privacy Policy URL** add করুন:
   ```
   https://auto-intel-privacy-policy.vercel.app
   ```
5. Data collection questionnaire fill করুন

**Declare করতে হবে:**
- Contact Info: Name, Email, Phone
- User Content: Photos, Files
- Identifiers: User ID, Device ID
- Usage Data: Product Interaction
- Diagnostics: Crash Data

### Step 4: Play Console এ Add করুন 🤖
1. **Play Console** এ login করুন
2. **Auto Intel** app select করুন
3. **App Content → Privacy Policy** এ যান
4. **Privacy Policy URL** add করুন:
   ```
   https://auto-intel-privacy-policy.vercel.app
   ```
5. **Data Safety** section complete করুন

**Declare করতে হবে:**
- Personal info: Name, Email, Phone
- Photos and videos: Yes
- Files and docs: Yes
- App activity: Yes
- Device or other IDs: Yes
- Location: No
